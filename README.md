# Batalla Naval

Proyecto académico del curso Fundamentos de Programación Orientada a Eventos.

## Descripción

Este proyecto implementará el juego Batalla Naval usando Java 17, JavaFX y Maven.

El desarrollo se realizará mediante ramas, commits y Pull Requests en GitHub.

## Tecnologías

- Java 17
- JavaFX
- Maven
- JUnit 5
- Git y GitHub

## Arquitectura propuesta

El proyecto se desarrollará siguiendo el patrón MVC:

- Modelo: lógica del juego, tablero, barcos, jugadores y reglas.
- Vista: archivos FXML, CSS y componentes JavaFX.
- Controlador: conexión entre la interfaz gráfica y la lógica del sistema.

## Estado actual

En este primer pull request solamente configuramos el proyecto base y se valida que JavaFX cargue una ventana inicial.
